# Examples

This page contains practical automation examples you can import and try immediately.

## Example 1: Hacker News Top Story

Extract the top story from Hacker News and take a screenshot.

```json
{
  "name": "Hacker News Top Story",
  "steps": [
    {
      "id": "1",
      "type": "navigate",
      "description": "Open Hacker News",
      "value": "https://news.ycombinator.com"
    },
    {
      "id": "2",
      "type": "wait",
      "description": "Wait for page load",
      "value": "1"
    },
    {
      "id": "3",
      "type": "extract",
      "description": "Get top story title",
      "selector": ".titleline a",
      "storeKey": "topStory"
    },
    {
      "id": "4",
      "type": "screenshot",
      "description": "Capture page",
      "savePath": "hn_top.png"
    }
  ]
}
```

**Result:** The top story title is saved in `topStory` variable and a screenshot is saved.

## Example 3: Product Price Monitoring

Check if a product price is below a threshold.

```json
{
  "name": "Price Monitor",
  "steps": [
    {
      "id": "1",
      "type": "setVariable",
      "description": "Set product URL",
      "variableName": "productUrl",
      "value": "https://example.com/product/12345"
    },
    {
      "id": "2",
      "type": "navigate",
      "description": "Open product page",
      "value": "{{productUrl}}"
    },
    {
      "id": "3",
      "type": "extractWithLogic",
      "description": "Check if price < $50",
      "selector": ".product-price",
      "condition": "lessThan",
      "expectedValue": "50",
      "transformType": "stripCurrency",
      "parseAsNumber": true
    }
  ]
}
```

**Result:** Returns whether the price is less than $50.

**Use case:** Set up a scheduler to run this automation periodically and notify you when price drops.

## Example 5: API Integration

Fetch data from an API and use it in automation.

```json
{
  "name": "API Data Extraction",
  "steps": [
    {
      "id": "1",
      "type": "apiCall",
      "description": "Fetch user data",
      "method": "GET",
      "url": "https://jsonplaceholder.typicode.com/users/1",
      "storeKey": "userData"
    },
    {
      "id": "2",
      "type": "navigate",
      "description": "Go to profile page",
      "value": "https://example.com/profile"
    },
    {
      "id": "3",
      "type": "type",
      "description": "Fill username",
      "selector": "input[name='username']",
      "value": "{{userData.username}}"
    },
    {
      "id": "4",
      "type": "type",
      "description": "Fill email",
      "selector": "input[name='email']",
      "value": "{{userData.email}}"
    }
  ]
}
```

**Result:** Fetches user data from API and fills form fields.

## Example 6: Multi-Page Navigation

Navigate through multiple pages and extract data.

```json
{
  "name": "Multi-Page Scraper",
  "steps": [
    {
      "id": "1",
      "type": "setVariable",
      "description": "Initialize page counter",
      "variableName": "currentPage",
      "value": "1"
    },
    {
      "id": "2",
      "type": "navigate",
      "description": "Go to page",
      "value": "https://example.com/products?page={{currentPage}}"
    },
    {
      "id": "3",
      "type": "extract",
      "description": "Get product count",
      "selector": ".product-count",
      "storeKey": "productCount"
    },
    {
      "id": "4",
      "type": "screenshot",
      "description": "Capture page",
      "savePath": "page_{{currentPage}}.png"
    },
    {
      "id": "5",
      "type": "modifyVariable",
      "description": "Next page",
      "variableName": "currentPage",
      "operation": "increment",
      "value": "1"
    }
  ]
}
```

**Note:** This example shows the concept. For actual looping, you'd need to wrap steps 2-5 in a loop construct (future feature).

## Example 7: Conditional Screenshot

Take a screenshot only if an element exists.

```json
{
  "name": "Conditional Screenshot",
  "steps": [
    {
      "id": "1",
      "type": "navigate",
      "description": "Open page",
      "value": "https://example.com"
    },
    {
      "id": "2",
      "type": "extractWithLogic",
      "description": "Check if banner exists",
      "selector": ".sale-banner",
      "condition": "contains",
      "expectedValue": "Sale"
    }
  ]
}
```

In your application code, check the `conditionMet` result before taking screenshot.

### Use Variables
